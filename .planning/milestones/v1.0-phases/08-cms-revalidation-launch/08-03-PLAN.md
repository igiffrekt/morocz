  <action>
**Create `src/components/ui/GoogleAnalytics.tsx`:**
- `'use client'` directive
- Props: `gaId: string`
- Uses `useEffect` to check if cookie consent has been granted:
  - Read `localStorage.getItem('cookie-notice-dismissed')` — if `'true'`, consent was given
  - If consent granted AND gaId is truthy, dynamically inject the gtag.js script:
    1. Create `<script async src="https://www.googletagmanager.com/gtag/js?id=${gaId}">` and append to document.head
    2. Create inline `<script>` with gtag config:
       ```
       window.dataLayer = window.dataLayer || [];
       function gtag(){dataLayer.push(arguments);}
       gtag('js', new Date());
       gtag('config', '${gaId}');
       ```
    3. Append to document.head
  - If consent NOT granted, do nothing (no GA script loads)
- Listen for a custom event `'cookie-consent-granted'` to load GA dynamically if the user accepts cookies during the session
- Also add `window.gtag` type declaration at top of file: `declare global { interface Window { dataLayer: unknown[]; gtag: (...args: unknown[]) => void; } }`
- Cleanup: remove script tags on unmount (optional — GA is typically permanent once loaded)

**Update `src/components/ui/CookieNotice.tsx`:**
- After `localStorage.setItem('cookie-notice-dismissed', 'true')` in `handleDismiss()`, dispatch a custom event: `window.dispatchEvent(new Event('cookie-consent-granted'))`
- This signals to GoogleAnalytics component that consent was just granted

**Update `src/app/layout.tsx`:**
- Import GoogleAnalytics from `@/components/ui/GoogleAnalytics`
- Read `process.env.NEXT_PUBLIC_GA_ID` and conditionally render:
  ```
  {process.env.NEXT_PUBLIC_GA_ID && (
    <GoogleAnalytics gaId={process.env.NEXT_PUBLIC_GA_ID} />
  )}
  ```
- Place it inside `<body>` before the closing tag, after CookieNotice
- Since this reads a NEXT_PUBLIC_ env var, it's available at build time — the component itself is client-side

**Update `.env.example`:**
- Add `NEXT_PUBLIC_GA_ID=` with comment `# Google Analytics 4 measurement ID (G-XXXXXXXXXX)`
