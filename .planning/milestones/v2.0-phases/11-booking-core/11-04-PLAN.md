**2. SessionStorage persistence for OAuth redirect (CRITICAL):**

The Google OAuth flow redirects the browser to Google and back to `/idopontfoglalas`. This remounts the component and loses all React state.

On every selection change, save wizard state to sessionStorage:
```typescript
const STORAGE_KEY = "morocz-booking-wizard";

// Save on change
useEffect(() => {
  sessionStorage.setItem(STORAGE_KEY, JSON.stringify({
    currentStep,
    selections,
    timestamp: Date.now(),
  }));
}, [currentStep, selections]);
```

On mount, check for saved state and restore:
```typescript
useEffect(() => {
  const saved = sessionStorage.getItem(STORAGE_KEY);
  if (saved) {
    try {
      const parsed = JSON.parse(saved);
      // Only restore if saved within the last 30 minutes
      if (Date.now() - parsed.timestamp < 30 * 60 * 1000) {
        setSelections(parsed.selections);
        // If returning from OAuth and user is now authenticated,
        // advance to step 4 (confirm) instead of step 3 (auth)
        if (parsed.currentStep >= 3) {
          setCurrentStep(parsed.currentStep);
        }
      } else {
        sessionStorage.removeItem(STORAGE_KEY);
      }
    } catch {
      sessionStorage.removeItem(STORAGE_KEY);
    }
  }
}, []); // Only on mount
```

On successful booking, clear sessionStorage:
```typescript
function handleBookingSuccess(bookingId: string) {
  sessionStorage.removeItem(STORAGE_KEY);
  setBookingResult({ bookingId, patientName, patientEmail });
  setCurrentStep(5); // Show success
}
```

**4. Soft hold timer (5-minute countdown):**

After the patient selects a time slot (completes Step 2), consider adding a POST to a hold endpoint. However, since the full hold mechanism adds complexity and the slot generation already filters held slots via GROQ query, a simpler approach: show a visual 5-minute timer starting when the patient enters Step 3 (auth). If the timer expires, show a message: "Az Ön által kiválasztott időpont foglalása lejárt. Kérjük, válasszon új időpontot." and return to Step 2.

Implement as a `useEffect` with `setTimeout`:
```typescript
const [holdExpired, setHoldExpired] = useState(false);
const holdTimerRef = useRef<NodeJS.Timeout | null>(null);

// Start timer when entering step 3 or 4
useEffect(() => {
  if (currentStep >= 3 && currentStep <= 4 && selections.selectedTime) {
    holdTimerRef.current = setTimeout(() => {
      setHoldExpired(true);
    }, 5 * 60 * 1000); // 5 minutes
    return () => {
      if (holdTimerRef.current) clearTimeout(holdTimerRef.current);
    };
  }
}, [currentStep, selections.selectedTime]);
```

If holdExpired becomes true, show a full-width banner with the expiry message and a "Vissza az időpontokhoz" button that goes to Step 2 and clears the selected time.
