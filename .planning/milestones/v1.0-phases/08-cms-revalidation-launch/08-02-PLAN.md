**Create `src/app/api/draft/route.ts`:**
- Export `GET` handler
- Read `secret` query param from `request.nextUrl.searchParams`
- Read `SANITY_PREVIEW_SECRET` from `process.env` — if not set, return 500
- Compare `secret` to `SANITY_PREVIEW_SECRET` — if mismatch or missing, return `NextResponse.json({ message: 'Invalid secret' }, { status: 401 })`
- On match: call `draftMode().enable()` (import `draftMode` from `next/headers`)
- Redirect to `request.nextUrl.searchParams.get('redirect') || '/'` using `NextResponse.redirect()`
- Import: `import { draftMode } from 'next/headers'` and `import { NextResponse, type NextRequest } from 'next/server'`

**Create `src/app/api/disable-draft/route.ts`:**
- Export `GET` handler
- Call `draftMode().disable()`
- Redirect to `request.nextUrl.searchParams.get('redirect') || '/'`
- Simple route, no secret needed (disabling preview is safe)

**Create `src/components/ui/DraftModeIndicator.tsx`:**
- Server Component (NO 'use client' — reads draftMode on server)
- Import `draftMode` from `next/headers`
- If `!draftMode().isEnabled` → return null
- Renders a fixed bottom bar (fixed bottom-0 left-0 right-0 z-[60]):
  - Dark background (bg-primary) with white text
  - Text: "Előnézet mód aktív" (Preview mode active) — centered, text-sm font-bold
  - A link to `/api/disable-draft` styled as a small button: "Kilépés" (Exit)
  - Padding: py-2 px-4, flex justify-center items-center gap-4
  - The "Kilépés" button: rounded-lg bg-white/20 hover:bg-white/30 text-white px-3 py-1 text-sm

**Update `src/sanity/lib/fetch.ts`:**
- Import `draftMode` from `next/headers`
- At the top of `sanityFetch`, check `const isDraft = draftMode().isEnabled`
- If isDraft:
  - Use a different client config: `useCdn: false` and add `token` from `process.env.SANITY_API_TOKEN`
  - Set `next: { revalidate: 0 }` (no caching in draft mode) — do NOT pass `tags` in draft mode
  - Create a separate draft client inline: `client.withConfig({ useCdn: false, token: process.env.SANITY_API_TOKEN, perspective: 'previewDrafts' })`
- If NOT isDraft: keep existing behavior exactly as-is (useCdn: true, tag-based revalidation)
- NOTE: `draftMode()` in Next.js 15 is synchronous and can be called in Server Components and Route Handlers. If the current Next.js version requires `await draftMode()`, make the function check accordingly. Check the import — in Next.js 15.x, `draftMode()` returns `{ isEnabled: boolean }` synchronously from `next/headers`.

**Update `src/app/layout.tsx`:**
- Import `DraftModeIndicator` from `@/components/ui/DraftModeIndicator`
- Add `<DraftModeIndicator />` right before the closing `</body>` tag (outside the MotionProvider, after CookieNotice area — it's a fixed-position bar so DOM position doesn't matter for layout)

**Update `.env.example`:**
- Add `SANITY_PREVIEW_SECRET=` with comment `# Secret for draft mode preview URL (share with editors)`
- Add `SANITY_API_TOKEN=` if not already present (needed for reading draft documents) with comment `# Sanity API token with viewer role (for draft mode)`
