**Props interface** (component receives data as props â€” never fetches from Sanity):

```typescript
interface ServiceCategoryData {
  _id: string;
  name?: string;
  emoji?: string;
  order?: number;
}

interface ServiceData {
  _id: string;
  name?: string;
  description?: string;
  icon?: SanityImageObject;
  category?: { _id: string; name?: string; emoji?: string };
  order?: number;
}

interface ServicesSectionProps {
  heading?: string;
  categories: ServiceCategoryData[];
  services: ServiceData[];
}
```

Note: `ServiceData.category` is an inline object (not a reference) because the GROQ query dereferences with `category->{_id, name, emoji}`. Import `SanityImageObject` from `../../../sanity.types`.

**State management:**
- `useState` for `activeCategoryId`, initialized to `categories[0]?._id` on first render
- Derive `filteredServices` by filtering `services` where `service.category?._id === activeCategoryId`

**Filter tabs:**
- Wrap in a horizontal-scrollable container: `overflow-x-auto` with `scrollbar-hide` (use `[&::-webkit-scrollbar]:hidden [-ms-overflow-style:none] [scrollbar-width:none]`)
- Each tab is a `<button>` with pill shape: `rounded-full px-5 py-2.5 text-sm font-semibold whitespace-nowrap transition-colors duration-300`
- Active tab: `bg-primary text-white` â€” show emoji before name text
- Inactive tab: `bg-white text-primary border border-gray-200 hover:bg-gray-50`
- No "All" tab â€” first category is active by default (per locked decision)
- Categories ordered by their `order` field (already sorted from GROQ query)
- Tab click calls `setActiveCategoryId(cat._id)`
- Flex row with `gap-2`

**Service cards grid:**
- Outer wrapper: `<div className="bg-white rounded-3xl p-6 md:p-10">` (white rounded container per decision)
- Grid: `grid grid-cols-2 lg:grid-cols-4 gap-4`
- Use `<AnimatePresence mode="popLayout">` wrapping the grid children for exit animations
- Each card wrapped in `<motion.div>` with:
  - `key={service._id}` (stable key from Sanity)
  - `layout="position"` (NOT `layout={true}` â€” prevents CLS per accumulated decision)
  - `initial={{ opacity: 0, scale: 0.95 }}`
  - `animate={{ opacity: 1, scale: 1 }}`
  - `exit={{ opacity: 0, scale: 0.95 }}`
  - `transition={{ duration: 0.5, ease: "easeOut" }}` (matches ~500ms calm pace per decision)

**Individual card styling:**
