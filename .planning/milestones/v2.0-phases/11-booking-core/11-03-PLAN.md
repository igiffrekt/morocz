- Today indicator: subtle border or dot

- On mount, fetch the weekly schedule and blocked dates from a lightweight endpoint or pass them as props from the page. - For the calendar highlighting, use the schedule data (days off + blocked dates) to gray out unavailable dates. The actual slot-level availability is fetched when a specific date is clicked.
