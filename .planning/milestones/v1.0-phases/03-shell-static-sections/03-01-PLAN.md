Create `src/components/layout/Header.tsx` as a "use client" component.

**Props interface:**
```ts
interface HeaderProps {
  logo?: SanityImageObject;
  clinicName?: string;
  navigationLinks?: Array<{ _key: string; label?: string; href?: string }>;
  phone?: string;
}
```
Import `SanityImageObject` from `../../../sanity.types` (or the project's sanity.types.ts root path). Import `urlFor` from `@/sanity/lib/image` for logo rendering.

**Scroll behavior (locked decision: shrink + shadow):**
- Use `useState` for `scrolled` boolean + `useEffect` with scroll listener
- Default header height: `py-5` (tall). Scrolled: `py-3` (shorter) + `shadow-md`
- Transition: `transition-all duration-300`
- Position: `fixed top-0 left-0 right-0 z-50 bg-surface-white`

**Desktop layout (md:768px+):**
- Logo on the left (render using `urlFor(logo).width(140).url()` if logo exists, fallback to clinicName text)
- Navigation links in the center as flat horizontal list: `flex gap-8`
- Each link: `<a href={link.href}>{link.label}</a>` with `text-text-light font-medium hover:text-primary transition-colors`
- CTA button on the right: "Foglaljon idopontot" (Hungarian for "Book an appointment")
  - Style: `bg-primary text-white px-6 py-2.5 rounded-xl font-semibold hover:bg-primary/90 transition-colors`
  - If phone prop exists, make the CTA an `<a href="tel:${phone}">`
- Container: `max-w-[88rem] mx-auto px-6`

**Mobile layout (below md:768px):**
- Logo on the left, hamburger button on the right
- Hamburger icon: three horizontal lines (use a simple SVG or three `<span>` bars)
- Navigation links and CTA button are HIDDEN on mobile (they go into MobileMenu)
- Pass `isOpen` and `onToggle` state to MobileMenu child component

**Important constraints:**
- "use client" directive required (scroll listener + state)
- Import from 'motion/react' is NOT needed here — use CSS transitions for the shrink effect (simpler, no animation library overhead for this)
- All text in Hungarian
- Touch-friendly: hamburger button minimum 44x44px tap target (RESP-03)
