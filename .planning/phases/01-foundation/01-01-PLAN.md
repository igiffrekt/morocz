**Font setup (`src/lib/fonts.ts`):**

Create a font configuration file that exports Plus Jakarta Sans from `next/font/google`:
```ts
import { Plus_Jakarta_Sans } from "next/font/google";

export const plusJakartaSans = Plus_Jakarta_Sans({
  subsets: ["latin"],
  weight: ["400", "500", "600", "700", "800"],
  variable: "--font-plus-jakarta-sans",
  display: "swap",
});
```

**Design tokens in Tailwind v4 (`src/app/globals.css`):**

Tailwind v4 uses CSS-first configuration. After the `@import "tailwindcss"` directive, add a `@theme` block defining all design tokens from the template:

```css
@import "tailwindcss";

@theme {
  /* Font family */
  --font-sans: var(--font-plus-jakarta-sans), "Plus Jakarta Sans", sans-serif;

  /* Colors — from template tailwind.config */
  --color-primary: #23264F;
  --color-secondary: #F4DCD6;
  --color-accent: #99CEB7;
  --color-yellow-card: #FAE988;
  --color-green-card: #A8DABC;
  --color-purple-card: #EABDE6;
  --color-blue-card: #8FB8FF;
  --color-background-light: #F2F4F8;
  --color-surface-white: #FFFFFF;
  --color-text-light: #1A1D2D;

  /* Border radius */
  --radius-DEFAULT: 1rem;
  --radius-xl: 1.5rem;
  --radius-2xl: 2rem;
  --radius-3xl: 2.5rem;

  /* Max width */
  --container-8xl: 88rem;
}
```

Note: Tailwind v4 auto-generates utility classes from `@theme` variables. `--color-primary` becomes `bg-primary`, `text-primary`, etc. `--radius-xl` becomes `rounded-xl`. Check Tailwind v4 docs if the exact variable naming convention differs — the key is that the template's color names (`primary`, `secondary`, `accent`, `yellow-card`, `green-card`, `purple-card`, `blue-card`, `background-light`, `surface-white`, `text-light`) are all available as Tailwind utilities.

Also add base styles after the theme block:
```css
body {
  @apply bg-background-light text-text-light antialiased;
}
```
